En el repositorio se encuentran los 4 problemas.
Para correr la parte 1 se  seleccionar la carpeta Parte 1, la cual tiene una main para cada algoritmo:(MainDijkstra, MainBellmanFord, MainFloydWarschall) Se debe ingresar el nombre del archivo que contiene el grafo. Se espera que cada linea de dicho archivo tenga la forma (nodo fuente, nodo destino, costo). Al usar el algoritmo de Dijkstra la salida dará el resultado el formato Destino/Predecesor/Costo con la respuesta. El algoritmo de BellmanFord 
da como salida el resultado desde el nodo fuente con el formatoDestino/Predecesor/Costo con la respuesta. El algoritmo FloydWarschall da por salida la matriz con los costos minimos entre pares.
Para correr la parte 2 se debe correr el MainBFS que pide la ruta dell archivo del grafo que espera que el archivo tenga en cada linea el formato de: vertice 1, vertice 2, peso. Y da por salida son los componentes conectados con el representante de su partición.
Para correr la  parte 3 se debe correr MainConstriccionesCalles, para esto se debe guardar el archivo con el grafo de las calles de la ciudad con el nombre de "distancespiloto.txt" en el que cada linea tiene la esquina1, la esquina2 y el costo. Da como salida las calles seleccionadas para ser doble vía.
Para correr la parte 4 se debe correr MainDistribucionLibros. La entrada debe tener la cantidad de libros por camion y bodega y un archivo con los camiones que representa los caminos de las fabricas a las bodegas y librerías y da como salida la cantidad máxima de libros que se pueden llevar.
